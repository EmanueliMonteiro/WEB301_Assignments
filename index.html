Toys R US

I didn't like the design in general, maybe because it's a children's site, it uses a lot of colors, which didn't look good. The navigation bar is fixed when we scroll down, so much of the screen is lost.

The Toys R Us website I reviewed implemented adaptive design. I liked more the mobile website than the desktop website. They used a hamburguer menu, it's more organized.

I liked the organization of the website. It's intuitive when you scroll down, and they separate their products by age.

For me the site has too many colors, which is not pleasing to the eye. Perhaps I would change the letter and spacing between the information.
Winners

All for you and your house.

The Winners website I reviewed implemented adaptive design. It is exactly the same as Marshalls website, as it is the same chain of stores. I couldn't find the search bar, which is weird as it's a primary thing for the user. There are internal links to visit websites that are related to the store.

The Winners website I reviewed implemented adaptive design. It doesn't use the hamburger menu, the website uses the scrollbar to the left to see the navigation menu. All sections are centered on the mobile screen, which is pleasant for the user.

It's the same as Marshalls. They only changed the color to black, but the design is responsive and everything is centralized. The search bar is in the center of the screen and easy to access.

I would probably increase the font size of the menu a bit, and maybe turn it into a hamburger menu.
Marshalls

All for you and your house.

The Marshalls website I reviewed implemented adaptive design. It doesn't use the hamburger menu, the website uses the scrollbar to the left to see the navigation menu. All sections are centered on the mobile screen, which is pleasant for the user.
