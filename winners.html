Winners

Analyze from a desktop point of view

The Winners website I reviewed implemented adaptive design. It is exactly the same as Marshalls website, as it is the same chain of stores. I couldn't find the search bar, which is weird as it's a primary thing for the user. There are internal links to visit websites that are related to the store.

Analyze from a mobile point of view

The Winners website I reviewed implemented adaptive design. It doesn't use the hamburger menu, the website uses the scrollbar to the left to see the navigation menu. All sections are centered on the mobile screen, which is pleasant for the user.

What I liked

It's the same as Marshalls. They only changed the color to black, but the design is responsive and everything is centralized. The search bar is in the center of the screen and easy to access.

What I would change

I would probably increase the font size of the menu a bit, and maybe turn it into a hamburger menu.
